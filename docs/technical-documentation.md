# Technical Documentation

## Project Overview

This project is a simple personal portfolio website developed using HTML, CSS, and JavaScript. 

The website includes multiple sections such as About Me, Projects, and Contact, and it demonstrates basic front-end development concepts.

## Project Structure

The project follows a simple and organized folder structure:

- index.html → Main webpage structure
- css/styles.css → Styling and layout design
- js/script.js → JavaScript functionality (time-based greeting)
- assets/images/ → Contains images used in the website
- docs/ → Documentation files
